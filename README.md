[![Build AMD64](https://github.com/hep-forge/qcdnum-feedstock/actions/workflows/autoupload.amd64.yml/badge.svg)](https://github.com/hep-forge/qcdnum-feedstock/actions/workflows/autoupload.amd64.yml)
[![Build ARM64](https://github.com/hep-forge/qcdnum-feedstock/actions/workflows/autoupload.arm64.yml/badge.svg)](https://github.com/hep-forge/qcdnum-feedstock/actions/workflows/autoupload.arm64.yml)

Feedstock for [qcdnum](https://www.nikhef.nl/~h24/qcdnum/index.html) — part of [hep-forge](https://anaconda.org/hep-forge).

QCDNUM is a very fast QCD evolution program written in FORTRAN77.

## Install

```bash
conda install -c hep-forge -c conda-forge qcdnum
```
